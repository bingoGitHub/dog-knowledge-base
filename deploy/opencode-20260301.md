### 🚀 安装与启动

在你的 WSL (Ubuntu) 终端中，直接运行以下命令即可全局安装 OpenCode：

```bash

npm install -g opencode-ai

```

等待命令执行完毕，没有任何报错，就说明安装成功了[](https://docs.coreshub.cn/console/big_model_server/call_scenario/opencode/)[](https://developer.aliyun.com/article/1687474)[](https://libraries.io/npm/@ex-machina%2Fopencode-darwin-arm64)。

安装好后，你可以直接在终端里输入以下命令来启动它：

```bash

opencode
```

### 🎯 启动后的关键步骤

OpenCode 启动后是一个全屏的终端交互界面（TUI）。为了能更好地使用，建议完成以下两步：

1. **初始化项目（推荐）**：进入你的项目目录后，在 OpenCode 界面中输入 `/init`。它会自动分析你的项目结构，并创建一个 `AGENTS.md` 文件，这样 OpenCode 就能更好地理解你的代码上下文了[](https://developer.aliyun.com/article/1687474)[](https://opencode.ai/docs/zh-tw/#%e4%bd%bf%e7%94%a8)[](https://apifox.com/apiskills/opencode-vs-claude-code-comparison/#%E5%9F%BA%E6%9C%AC%E4%BD%BF%E7%94%A8)。
    
2. **配置 AI 模型**：OpenCode 本身不提供 AI 能力，需要连接一个大型语言模型（如 OpenAI 的 GPT、Anthropic 的 Claude 等）。你需要在界面中输入 `/connect` 命令，并按照提示选择一个模型提供商，然后输入你的 API 密钥[](https://developer.aliyun.com/article/1687474)[](https://opencode.ai/docs/zh-tw/#%e4%bd%bf%e7%94%a8)[](https://apifox.com/apiskills/opencode-vs-claude-code-comparison/#%E5%9F%BA%E6%9C%AC%E4%BD%BF%E7%94%A8)。
    

### 💡 几个有用的命令

进入 OpenCode 界面后，这几个命令能帮你更快上手：

|命令|功能说明|
|---|---|
|**Tab 键**|在 `build`（修改代码）和 `plan`（只出方案）两种模式间切换[](https://developer.aliyun.com/article/1687474)[](https://opencode.ai/docs/zh-tw/#%e4%bd%bf%e7%94%a8)[](https://apifox.com/apiskills/opencode-vs-claude-code-comparison/#%E5%9F%BA%E6%9C%AC%E4%BD%BF%E7%94%A8)。|
|**/models**|切换或查看当前可用的 AI 模型[](https://developer.aliyun.com/article/1687474)[](https://cloud.tencent.com/developer/article/2628525)。|
|**/undo**|如果 AI 修改的代码不满意，可以用这个命令一键撤销[](https://developer.aliyun.com/article/1687474)[](https://opencode.ai/docs/zh-tw/#%e4%bd%bf%e7%94%a8)。|
|**/share**|生成一个分享链接，把你的对话记录分享给同事[](https://developer.aliyun.com/article/1687474)[](https://cloud.tencent.com/developer/article/2628525)。|
